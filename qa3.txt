Ошибка в приведенном примере заключается в отсутствии возврата результата выполнения метода sum в декораторе.
Для исправления кода необходимо вернуть значение полученное в результате выполнения функции/функтора f в коде декортатора wrapper.

Также решением проблемы может быть удаление аннотации @_log перед методом sum. Это позволит получать корректные значения в результате вызова метода.

Дополнительно целесообразно перед журналированием во враппере добавить проверку уровня журналирования. В случае его смены на менее подробные (INFO, WARN .. ) это позволит сократить количество вызовов форматирования строки. Предлагаю добавить следующую проверку:

if logger.isEnabledFor(logging.DEBUG):
	logger.debug('%s :: args: %s, kwargs: %s',
				f.__name__, args, kwargs)
